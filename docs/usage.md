# Usage guide

_For site builders and content editors._

## Adding the block

1. Edit a post or page.
2. Click the **+** inserter and search for **Logo Carousel** (it's in the
   **Media** category).
3. Click **Media Library** and select the logos you want to show. You can
   select several at once, and reorder or change them later.

To change the selection afterwards, click the block and use **Edit images** in
the block toolbar.

## Settings

Open the block's settings panel (the cog icon) to find these controls.

### Carousel settings

| Setting | What it does |
| ------- | ------------ |
| **Scroll speed (seconds per logo set)** | How long one set of logos takes to glide past. Lower is faster. The speed stays the same however many times the set repeats. |
| **Pause on hover** | Stops the scroll while the visitor hovers over the strip. |
| **Logo height (px)** | The display height of every logo. Widths scale automatically to keep each logo's proportions. |
| **Gap between logos (px)** | The space between logos. |
| **Greyscale logos** | Shows logos in greyscale, returning to full colour when hovered. Handy for a uniform, understated strip. |

### Advanced

| Setting | What it does |
| ------- | ------------ |
| **Repeat logo set** | How many copies of your logos fill the strip. Increase this if you see a gap appear before the loop repeats. |

## Tips

- **Use enough logos.** A continuous loop looks best with roughly six or more
  logos. With fewer, raise **Repeat logo set** (Advanced) so the strip stays
  filled and the loop stays seamless.
- **Transparent logos work best.** PNG or SVG logos with transparent
  backgrounds sit cleanly on any page colour. Combine with **Greyscale** for a
  tidy, consistent look.
- **Wide and full width.** Use the block's alignment toolbar to make the strip
  **Wide** or **Full width** for an edge-to-edge band.
- **Reduced motion.** Visitors who have asked their device to reduce motion
  will see the logos as a static, centred row rather than a scrolling strip —
  this is automatic and needs no setup.

For visual customisation beyond these settings, see the
[styling guide](styling.md).
